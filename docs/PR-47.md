# PR 47: Bounded internal-structure reconnaissance for RAT_BACK-style large-PL8

## Summary
This PR adds bounded deterministic reconnaissance for RAT_BACK-style structured large-PL8 payloads without implementing final decode.

Implemented scope:
- Reused the shared 24-byte large-PL8 outer header parser.
- Added `probe_caesar2_rat_back_structured_pl8_regions(...)` to inspect payload internals in a bounded way.
- Added deterministic report formatting for raw leading fields, record/table hints, and candidate region summaries.
- Added deterministic comparison support for two structured-region probe reports.
- Added CLI commands:
  - `--probe-pl8-structured-regions <path>`
  - `--compare-pl8-structured-regions <lhs_imagepl8> <rhs_imagepl8>`
- Added tests for stable report formatting, deterministic hint behavior, and malformed/truncated payload failure.

## Reconnaissance behavior (bounded)
The new probe:
1. Parses width/height from the confirmed 24-byte outer header.
2. Requires at least 8 payload bytes after the outer header.
3. Samples early payload fields as little-endian `u16` and `u32` values.
4. Scans bounded fixed record sizes (`4, 6, 8, 12, 16`) for repeated-structure indicators and plausible `(offset,length)` pairs.
5. Produces candidate region hints from:
   - adjacent leading `u32` pairs interpreted as `(start,size)`, and
   - monotonic offset-table segmentation from leading offsets.
6. Emits `decode_status: not_attempted` explicitly.

No UI/game semantics are assigned to unknown fields.
No generic PL8 framework is introduced in this PR.
